Migração: dados antigos → mesas/{mesaId}

Essa página existe só pra rodar UMA VEZ, depois de atualizar o site pra versão com mesas separadas. Antes dela, toda a rede (fichas, npcs, calendário, log de dados etc.) vivia na raiz do banco. Agora cada Mestre tem sua própria mesa em mesas/{mesaId}/....

Este botão copia TUDO que hoje está na raiz do banco (exceto itensGlobais, que continua compartilhado de propósito) para dentro de mesas/frkstnmsk, e cria o loginIndex de cada ficha já existente apontando pra essa mesa. Ajuste o id da mesa de destino no campo abaixo se o dono original das fichas não for o frkstnmsk.
